
Autor: Emmanuel Martínez Guerrero

La idea de este proyecto se basa en la seguridad de archivos o información. Dado que hay diversas formas o métodos con los que se pueden resguardar varios tipos de archivos, yo opte por desarrollar un programa de encriptado principalmente orientado a archivos de texto.

El funcionamiento en forma general del programa consiste por una parte en extraer cada caracter de una fuente de texto (.txt) e interpretar el valor (DEC) que tiene cada caracter en el lenguaje ASCII. Una vez obtenido este valor, se le aplica una operación matematica con el fin de que este caracter cambie de valor en el lenguaje ASCII. Con el valor nuevo de cada caracter se genera otro archivo de texto el cual queda de forma encriptada.

Por otra parte el programa ha sido diseñado para realizar el proceso inverso de desenciptar mediante un proceso analogo al de encriptado. En este proceso se toma un archivo fuente (.txt) el cual ha sido previamente encriptado, para posteriormente aplicarle el proceso inverso de encriptación y finalmente obtener un nuevo archivo de texto con el contenido desencriptado.


Finalmente, dentro de la carpeta de cada programa se dan algunas instrucciones y consideraciones para el optimo funcionamiento de cada programa.
